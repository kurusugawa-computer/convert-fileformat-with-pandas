# Requirements
* Python 3.6+

# Install

```
$ pip install convpandas
```

https://pypi.org/project/convpandas/


# Usage

## csv2xlsx
Convert csv file to xlsx file.

```
$ convpandas csv2xlsx in.csv out.xlsx
```

## xlsx2csv
Convert xlsx file to csv file.

```
$ convpandas xlsx2csv in.xlsx out.csv
```
